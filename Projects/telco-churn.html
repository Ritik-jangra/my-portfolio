Telco Churn Prediction

Live Demo GitHub Repo

Project Description

This project predicts customer churn using the Telco dataset. A Random Forest model was trained to classify whether a customer is likely to leave based on various features such as tenure, contract type, monthly charges, and more.

Tech Stack

  • Python
  • Streamlit
  • Random Forest Classifier
  • SHAP for Model Explainability
  • Pandas, Sklearn

SHAP Explanation

  • SHAP values provide feature-wise contribution to the model's prediction.
  • Visualizes how each feature increases or decreases the likelihood of churn.
  • Used `shap.plots.waterfall()` for individual prediction explanations.

Features

  • Interactive Streamlit UI for user input and predictions
  • Explains predictions using SHAP plots
  • Trained on cleaned Telco Churn dataset
  • Live deployment on Streamlit Cloud

Performance Metrics

  • Accuracy: 82%
  • Precision: 78%
  • Recall: 76%
  • F1-Score: 77%

Challenges & Learnings

  • Data preprocessing required deep cleaning due to categorical encoding issues
  • Learned effective use of SHAP for black-box model interpretation
  • Improved understanding of deployment workflow using Streamlit
